# Hosted paper PDFs

Drop clean, main-text PDFs here (no highlights/annotations, no extended data or
supplements) and the "PDF" links on `publications.html` point at them.

**Naming:** `<kebab-title-slug>-<year>.pdf` — e.g.
`compact-transcriptional-effectors-2024.pdf`. Keep it lowercase, no spaces.

**What's safe to host:** open-access papers (Nat Commun, PLoS, Genome Biology,
NAR, bioRxiv preprints) are freely licensed — host the publisher PDF directly.
For subscription journals (Cell, Nature, Nat Biotech/Methods, Science, Mol Cell,
Cell Systems, CRISPR J, Hum Gene Ther), host a clean self-archived copy; the
typeset publisher PDF is technically the publisher's.

Until a paper's PDF is added here, its "PDF" link falls back to the free full
text on PubMed Central.
